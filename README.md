## 🚀 Run Locally (Step by Step)

### Step 1 — Make sure Python is installed
```bash
python --version
# Should show Python 3.9 or higher
```

### Step 2 — Install required libraries
```bash
pip install streamlit pandas plotly anthropic
```

### Step 3 — Run the app
```bash
streamlit run app.py
```

Your browser will open automatically at **http://localhost:8501** 🎉

## 🌐 Deploy for FREE (Share with Anyone)

### Option A — Streamlit Cloud (Easiest, Recommended)

1. Create a free account at **https://streamlit.io**
2. Push your code to GitHub (also free):
   ```bash
   git init
   git add .
   git commit -m "FinSight AI first version"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/finsight-ai.git
   git push -u origin main
   ```
3. Go to **share.streamlit.io** → New app → Select your repo → Deploy
4. You get a free link like: `https://ibrahim-finsight.streamlit.app` ✅

### Option B — Run on your phone's hotspot for demos
```bash
streamlit run app.py --server.address=0.0.0.0
```
Anyone on the same WiFi can access it via your IP address.
